

*) Install Kamene library for reading pcap files. 
- pip3 install kamene

or 

- py -3.x -m pip install kamene

*) Install scikit-learn

- pip3 install scikit-learn

or

- py -3.x -m pip install scikit-learn

Install Tensorflow
- py -3.x -m pip install tensorflow
Install Keras
- py -3.x -m pip install keras



If an error comes back that python doesn't have module "install", then download get-pip.py, and install pip for the version of python you're using. 

- sudo py -3.x get-pip.py

3.x is the version of python you wish to run this program under.
